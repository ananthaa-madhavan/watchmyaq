Design Process

Project Origin

This project began with the goal of making air quality data more accessible, visual, and meaningful for everyday use rather than purely numerical reporting.

System Overview

BreathSafeSTL combines environmental particulate sensors, microcontroller-based data collection, and a web-based visualization system to map air quality in real time.

Development Process

The system was built iteratively, starting from basic sensor testing, then moving into data logging, and finally evolving into a structured visualization platform with planned live mapping integration.

Iteration & Improvements

Over time, the design improved through calibration of sensor readings, refinement of data handling, and simplification of the user interface to make environmental information easier to interpret.
